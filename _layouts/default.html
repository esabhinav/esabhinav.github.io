1. Web Application for TamilNadu Bharat Petroleum Dealer Association, Java

          Developed a complete web application for TamilNadu Bharat Petroleum Dealer Association using Java, SQL, JSP and JQuery for development. The pages were hosted from Amazon Web Services Cloud. The application is aimed to connect the Bharat Petroleum dealers across Tamil Nadu and ease their work by keeping track of certain tasks.
Tamil Nadu Bharat Petroleum Dealer Association


2. Home Automation With Raspberry Pi, Java

          Implemented a home automation system which enabled controlling remotely connected devices through Raspberry Pi. The project used client-server paradigm where the Raspberry acted as the server controlling all devices. A WebUI was also developed to keep track of connected devices. The project helped understand the nuances of Internet of Things.


3. Predicting Stock Prices using Machine Learning, Python

          Implemented various Machine Learning models to predict the rise and fall of stock prices before the actual event of an increase or decrease in the stock price occurs. The project involved various stages like data pre-processing, feature engineering, data-modeling and evaluation. Multiple regression models like Linear Regression, Logistic Regression and Decision Trees were tried out as part of the project.
